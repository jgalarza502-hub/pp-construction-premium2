# P&P Construction Premium Website

A complete Next.js website using P&P Construction's actual project photos and contact information.

## Run it on your PC

1. Extract this ZIP.
2. Open the extracted folder in VS Code.
3. Open **Terminal → New Terminal**.
4. Run:

```powershell
npm install
npm run dev
```

5. Open `http://localhost:3000`.

## Upload to GitHub

1. Sign in to GitHub and click **New repository**.
2. Name it `pp-construction` and create it as a **Public** repository.
3. In VS Code, open the Source Control tab and choose **Publish to GitHub**, or upload the extracted files through GitHub's **Add file → Upload files** option.
4. In Vercel, choose **Add New → Project**, import the GitHub repository, and click **Deploy**.
